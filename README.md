# POC: Storybook React

It demonstrates the minimal configuration for [Storybook](https://github.com/storybookjs) targeting [React](https://github.com/facebook/react).

## How to run

| Description | Command |
| :--- | :--- |
| Install dependencies | `npm install` |
| Run tests | `npm test` |
| Run app | `npm start` |
| Run Storybook | `npm run storybook` |
| Build app | `npm run build` |
| Build Storybook | `npm run build-storybook` |
